## Installation

Setup spark by running:
```bash
bin/installSpark.sh
```

Install python3:
```bash
sudo apt install python3 python3-pip
```

Install python3 libs:
```
pip3 install jupyter py4j pyspark
```

## Launching

To launch run:
```bash
jupyter notebook
```
